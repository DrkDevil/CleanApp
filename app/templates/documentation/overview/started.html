Requirements

BonsaiApp only has a few software and knowledge requirements. It is required you have some basic knowledge of shell, and commands. To learn about git and terminal youcan read more about here.

GIT, NPM, *BOWER, Gulp.js, and Terminal or your prefered CLI interface.

* After the release of Beta we will be removing BOWER and replacing it with an NPM only solution

How to Install via shell

A shell is the perfered method of downloading and installing BonsaiApp. Simply becasue even after you download BonsaiApp manually you will still need to use it to install all the applications dependencies, as well as run other commands listed in the CLI Commands Page.

Depending on your operating system, there are multiple shells out there you can use. It is really all a matter of your preference.

Below is a list of some of the more commonly used shell programs.

  • Termnal - on Mac OSX
  • GitBash - on Windows
  • Command Prompt - on Windows
  • Linux Terminal - on Linux

Installation with terminal is a breaze. From withing your cli terminal just enter the command:

$ git clone http://github.com/drkdevil/CleanApp.git

After you have clone the repo you need to open the directory you downloaded it into from within terminal and run the command:

$ npm install
