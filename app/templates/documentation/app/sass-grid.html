Sass Grid System

What is a grid system?

The Grid System.

The Grid System is the brick and mortar of your applications layout. BonsaiUI requires it when using columns to layout your pages.

There are two types of grids you can use.

The .grid class will create you a responsive grid that is set at a fixed-width. This grid will resize itself depending on the breakpoints you set in the variables.scss The other grid you can use is the .grid-fw class. This stands for full-width because the grid will always be set at a width of 100%.

While containers can be nested, most layouts do not require a nested container.
