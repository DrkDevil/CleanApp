Alerts & Notifications

Alerts provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages. Alerts are available for any length of text or HTML, as well as an optional dismiss button.

The .alert Class

The .alert class is the base class that creates your default unstyled alerts. It is designed for use on <div> tags.

Alternative Colors & Links

To add a color class simply add -primary, -info, or another color class listed in the variables styleguide directly to the .btn class. This will result in one class that looks like this .alert-primary.

The alert border is a transparent shade that matches the background color you choose automatically.
