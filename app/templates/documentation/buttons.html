Buttons

The <button> tag defines a clickable button. Inside a <button> element you can put content, like text or images. Styling a <button> element is as easy as adding the .btn class. Further customization can be added to style your butons as decibed in the documentation below.

The .btn Class

The .btn class is the base class that creates your default unstyled buttons. It is designed for use on <button> elements. However, you can also use it on other elements such as <div> or <a>.

To add a color class simply add -primary, -info, or another color class listed in the variables styleguide directly to the .btn class. This will result in one class that looks like this .btn-primary
